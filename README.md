### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/abhishekc98/memory-game-vanilla-js.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run locally
    ```sh
    npm run dev
    ```
